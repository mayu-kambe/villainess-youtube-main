# 構成（全8パート）

執筆前の**詳細構成案**の出し方は [detailed-outline.md](detailed-outline.md) を参照。本文執筆は構成案のユーザー承認後に開始する。

## 🎬 プロローグ：この物語の結末（掴み）【300〜400字】

**目的**：クリックした視聴者に「どんな動画か」を1発で伝える。

```
① サムネシーン再現（3つのセリフをダイジェスト）
  → 右サムネ①：元婚約者の胸糞セリフ
  → 右サムネ②：ヒロイン悪役の胸糞セリフ
  → 左サムネ：悪役令嬢の逆転セリフ
  → 暗転

② 悪役令嬢の独白（4〜5行）
  → 自己紹介（名前・身分・何者か）
  → 裏で支えていたことの示唆
  → 理不尽の要約（「〇〇と言われ続けました」）
  → フック（「これは、そんな私が――〇〇した物語ですわ」）
```

## 🩰 第1章：悪役令嬢の日常【600〜900字】

**目的**：主人公の有能さ＋認められない理不尽を見せる。

```
□ 主人公が自分の仕事＋元婚約者の仕事を二人分こなしている描写
□ 元婚約者が主人公の成果を自分の手柄だと思い込んでいる描写
□ 元婚約者が主人公を見下す態度（感謝ゼロ・命令口調）
□ ヒロイン悪役の登場を予告する一言
```

---

## ⚔️ 第2章：敵役登場【1,200〜1,500字】

**目的**：ヒロイン悪役の二面性を見せ、視聴者のヘイトを溜める。

```
□ ヒロイン悪役の「表の顔」を男の前で見せる
□ 主人公と二人きりになった時の「裏の顔」を見せる
□ 元婚約者がヒロイン悪役を優遇し、主人公を冷遇する対比
□ 元婚約者が主人公の成果を自分の功績として発表するシーン
□ 周囲の人物が「本当は主人公のおかげ」と気づいている伏線
```

## 💋 第3章：右サムネのシーン（胸糞MAX）【1,000〜1,500字】

**目的**：視聴者の怒りを最大にする。フラストレーションの頂点。

```
□ 右サムネ①：元婚約者の最も酷い一言（前後に間を作る）
□ 右サムネ②：ヒロイン悪役の最も酷い一言（元婚約者の背後or横で）
□ 主人公は表情を崩さない（怒りを内に秘める）
□ 主人公の独白で「裏で支えていた事実」を視聴者に再確認させる
□ 周囲の噂・同情で主人公の孤立感を強調
```

## 🔥 第4章：新相手の登場＋決意【800〜1,200字】

**目的**：新相手がちょろっと登場して助言。主人公が自分で立ち上がる。

```
□ 新相手は別の場面（社交場・会議・偶然の出会い）で自然に登場
□ 核心を突く助言を一つだけ（「自分を安く扱う場所にいる必要はない」的な）
□ 新相手が主人公の「裏の貢献」を知っている（情報力の高さを示す）
□ 去り際にさりげない気遣い（好意）→ 主人公は「仕事上の判断」と解釈
□ 新相手が振り返る・足を止める等の仕草 → 主人公は気づかない、視聴者だけが見る
□ 主人公が自分の意志で決意する（新相手に言われたからではなく自分で気づく）
□ 決意の瞬間を印象的なセリフ＋象徴的動作で（鏡の前、手袋をはめる等）
```

**新相手のセリフ指針**：
- 設定ファイルの「名台詞集」を参考にしつつ、ビジネス文脈に自然に変換
- 好意は1〜2箇所だけ仕込む（やりすぎない）
- 主人公が「仕事の話」と解釈できる二重の意味を持たせる

## 💞 第5章：左サムネのシーン（反撃＋退場）【2,500〜3,500字・最長パート】

**目的**：視聴者が一番見たいシーン。丁寧に、長く。

### 第5章「制裁パート」の推奨フロー（スカッと密度アップ）

**起点**：制裁の本編は **左サムネ展開＝1回目の反撃シーンからスタート**してよい。

```
■【制裁①】1回目の反撃（左サムネ）
□ 左サムネのセリフ・動作で切り込む
□ 悪役は「泣く／勝った」と思い込んでいる → まさかの展開に動揺する

■【制裁②】2回目の反撃（追い打ち）
□ ご精算・契約・証拠を畳み掛ける
□ 「裏で支えていた事実」の公開暴露（資料・筆跡・成果ゼロの証明）
□ ヒロイン悪役の二面性を公開暴露（証言・泣きは通用しない）
□ 悪役の胸糞セリフをブーメランにする

■【悪役の抵抗①】逆ギレ・責任転嫁・仲間割れ（不様）

■【悪役の抵抗②】最後の悪あがき（脅し・すすり泣き 等）

■【主人公のトドメ】※必ず主人公本人。倍返しは本人の口から。

■【無様な末路の始まり】命乞い・復縁要求（第5章内に収めてもよい）

■【新相手の登場】（任意）**主人公のトドメのあと**がおすすめ
```

**従来チェックリスト（併用）**：

```
□ 公の場に主人公が登場（会議・パーティー・式典）
□ 退職宣言 or 離脱宣言 or 婚約破棄への反撃
□ 左サムネが「1回目の反撃」の核になっている
```

## 🌈 第6章：崩壊＋ザマァ【3,000〜4,000字】

**目的**：悪役たちの因果応報。ヒロイン悪役への直接対決。

**※第5章で夜会の制裁・命乞い・新相手登場まで描いた場合**、第6章は時間経過後の崩壊・別対決・独白に振るとテンポが良い。

### ■ 崩壊フェーズ（時間経過で段階的に）

```
□ 1週間後：現場が回らない。主人公がやっていた業務が止まる
□ 2週間後：社交界での噂。元婚約者の評判が暴落
□ 3週間後：元婚約者が自分の無力さに気づく
  → 「俺は、この仕事をどうやっていた……？」の絶望
```

### ■ ヒロイン悪役との直接対決

```
□ ヒロイン悪役が主人公のもとに乗り込んでくる（乱れた姿）
□ 仮面が完全に剥がれる（口調が変わる・本性が出る）
□ ヒロイン悪役の本音（境遇への恨み・嫉妬を叫ぶ）
□ 主人公が一瞬だけ同情を見せる → だが揺るがない
□ 止めの一撃：「あなたに足りなかったのは、〇〇ではなく△△」
□ ヒロイン悪役を突き放す（「それを自分で考えることが〜」）
```

### ■ 元婚約者の命乞い・後悔

```
□ 第5章で未描画なら：頭を下げて「戻ってきてくれ」
□ 主人公が相手の胸糞セリフを引用してブーメラン
□ 主人公が成果を見せる（却下された案を自分で実行して成功）※省略可
□ 決め台詞で完全論破
□ 婚約にも触れる → 冷ややかに拒否
□ 第5章で命乞いを描いた場合：独白・没落の静かな後悔に寄せる
```

## エピローグ：新しい日常＋じれったさ【1,500〜2,000字】

**目的**：悪役の末路を語り、主人公の幸せを見せ、次回への期待を作る。

### ■ 悪役の末路（ナレーション＋小シーン）

```
□ 元婚約者：罷免・降格・社会的信用の崩壊
□ ヒロイン悪役：居場所を失う → ただし「はじめの一歩」を匂わせる描写
```

### ■ 主人公の新生活

```
□ 本来の場所で生き生きと働いている
□ 侍女・側近との温かい会話
```

### ■ 新相手の再登場（じれったさ＋キュンキュン）

```
□ ビジネスを口実に訪問
□ さりげない差し入れ（主人公の好みを知っている）
  → 主人公は「試供品」「手土産」と合理的に解釈
□ 好意がにじむセリフ → 主人公は「仕事の話」と解釈
  → 二重の意味を持つ一言（視聴者にだけ真意が伝わる）
□ 新相手の表情・仕草に本音が漏れる → 主人公は気づかない
□ 侍女が視聴者の代弁をする
  → 「なぜお好みをご存知なのでしょう？」
  → 主人公「〇〇ですもの。当然ですわ」で流す
  → 侍女（小さく笑って）「……本当に、それだけでしょうか」
```

### ■ 締めの独白

```
□ 視聴者へのメッセージ（物語のテーマを一言で）
□ 「それでは皆様、次の動画でお会いしましょう。ごきげんよう。」
```

**じれったさ設計のルール**：
- 新相手の好意は**視聴者にだけ**伝わり、主人公は気づかない
- 主人公は全てを合理的・ビジネス的に解釈する
- 侍女・側近が「視聴者の代弁者」として疑問を投げかける
- 両思い未満で止める（次の動画への引き）
- 設定ファイルの「名台詞集」「弱点」「関係の深まり方」を活用して、その主人公・新相手ペアならではのじれったさを作る
