# 詳細構成案（必ず最初に提示）

執筆依頼を受けたら、**本文のプロローグを書く前に**、ユーザーへ **【詳細構成案】** を出し、**承認後**に執筆に入る。

## AIが最初に出力すること

1. **一行あらすじ**（50〜100字程度）
2. **視聴者の感情の流れ**（プロローグ→エピローグを1行ずつ）
3. **全8パート対応表**（各パートごとに必ず記載）

| 書く項目（各パート） | 内容 |
|---|---|
| **場所・時間** | 主な【場面】 |
| **目的** | その章で視聴者に起こしたい感情 |
| **起きること** | 主要イベントを箇条書き（3〜7個） |
| **登場人物** | その章に出すキャラ |
| **サムネ・決め台詞の回収** | 右①②・左サムネのどれをどこで使うか |
| **伏線・回収** | 前章からの継続／この章で仕込む伏線 |
| **第5章のみ** | 制裁①②・悪役の抵抗・主人公のトドメ・命乞い・新相手登場の**順序** |

4. **第5章（制裁）の段階メモ**  
   [plot-structure.md](plot-structure.md) の「第5章・制裁パートの推奨フロー」に沿った箇条書き。

5. **参考動画の文字起こしを貼っている場合**  
   元テキストのシーン → 各パートへの**割り当て**を併記。語句のコピペはせず、骨格とテンポのみ参考。

## 出力フォーマット例（見出し）

```markdown
## 【詳細構成案】（タイトル：〇〇）

### 一行あらすじ
（本文）

### 感情の流れ（視聴者）
- プロローグ：……
（エピローグまで）

### パート別詳細
#### プロローグ
- 場所・時間：……
- 目的：……
- 起きること：……
（第6章・エピローグまで同様）

### 第5章・制裁パートの段階（該当する場合）
1. 制裁①：……
2. …

### 参考動画からの割り当て（該当する場合）
……
```

## ユーザー側の合図

- **「構成OK」「この構成で執筆して」** → プロローグから執筆開始
- **「第○章だけ直して」** → 該当部分を修正した構成案を再提出
- 構成案の段階では**セリフ本文は書かない**（必要なら例文は1〜2行まで）

## 参考：文字起こしリライト時の追加指示

- 本文を書く前に、必ず上記【詳細構成案】を出す。構成案に「シーン分割 → 各パートへの割り当て」を含める。
- ユーザーが構成案にOKするまで、プロローグから執筆に入らない。
- 著作権のため、**語句のコピペはせず**、同趣旨を悪役令嬢世界観の独自台詞に置換する。

詳細はリポジトリの `プロットテンプレート【確定版】.md` と同期している。
