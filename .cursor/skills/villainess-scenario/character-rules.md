# キャラ設定ファイルの読み取りルール

添付された設定ファイルから、以下を自動で読み取ってシナリオに反映する。

## 主人公（悪役令嬢）から読み取る項目

- 名前・身分・家名・社交界での呼称
- 武器（知性／魔法／剣術／会計／情報 等）
- 断罪スタイル（氷の理性／笑顔のまま／剣で語る 等）
- 口癖・決め台詞のパターン → 第5章・第6章の決め台詞に活用
- 性格（表層と深層） → 二面性の演出に使う
- 弱点（甘い菓子、涙もろい等） → エピローグのじれったさポイントに使う
- 象徴アイテム（扇子／手袋／剣 等） → 演技指示で繰り返し使う
- 侍女・側近の名前と関係性 → エピローグの代弁者に使う
- 裏で支えていた仕事の内容 → 第1章・第5章の暴露に使う

## 新相手から読み取る項目

- 名前・爵位・異名
- 性格（表層と深層）
- 主人公との関係の始まり方 → 第4章の登場シーンに反映
- 好意の表し方の特徴 → エピローグのじれったさシーンに反映
- 名台詞集 → ビジネス文脈に自然変換して使う
- 弱点・趣味 → 主人公との対比やキュンポイントに使う

## ヒロイン悪役から読み取る項目

- 表の顔・裏の顔の切り替えパターン → 第2章で見せる
- 男性攻略の手口 → 第2章〜第3章の具体的描写に使う
- 弱点・崩れ方 → 第5章・第6章の崩壊に使う
- 断罪された時の反応 → 第6章の直接対決に使う

## 元婚約者から読み取る項目

- 立場・肩書・社会的評価
- 欠点・勘違いの内容 → 第1章〜第3章の胸糞描写に使う
- 自覚のタイミングと反応 → 第6章の崩壊に使う
