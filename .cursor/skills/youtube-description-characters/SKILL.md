# 悪役令嬢系YouTube動画 概要欄用 登場人物紹介スキル

## 役割

悪役令嬢系スカッと恋愛アニメ動画の概要欄に貼り付ける、視聴前のワクワク感を煽る短い登場人物紹介を生成する。

「ネタバレせず、キャラの立ち位置と魅力で釣る」が至上命題。

---

## 進め方（厳守）

1. ユーザーから対象動画（シナリオ）を確認する
2. 該当シナリオファイルと、登場4キャラの設定ファイルを読み取る
3. 下記フォーマットに沿って4人分の紹介文を作成
4. ユーザーに内容を提示 → 承認後、`概要欄/【主人公名①】登場人物紹介.md` として保存

---

## 必須情報の収集

シナリオファイルの冒頭から以下を抽出：

- **主人公**（悪役令嬢）：名前・爵位・異名
- **元婚約者**：名前・肩書・異名
- **ヒロイン悪役**：名前・爵位・特徴
- **新相手**：名前・爵位・異名

その上で各キャラ設定ファイル（`キャラクターたち/` 配下）を読み、
年齢・象徴的な肩書・キャラの本質を1行で表現できる要素を拾う。

---

## 出力フォーマット【厳守】

````markdown
# 【主人公名①】動画タイトル ／ 概要欄用 登場人物紹介

```text
━━━━━━━━━━━━━━━━━━━
　登場人物
━━━━━━━━━━━━━━━━━━━

◆ [主人公フルネーム]（[年齢]歳／[爵位]）
　[家柄や立場を1行]。社交界では「[異名]」と呼ばれる[象徴]。
　[物語フックの1行――ネタバレせず、これから何かが起きそうな含み]。

◆ [元婚約者フルネーム]（[年齢]歳／[肩書]）
　[主人公との関係]。[出自や異名を1行]。
　[人物の歪み・違和感を1行で示唆]。

◆ [ヒロイン悪役フルネーム]（[年齢]歳／[爵位]）
　[ビジュアル特徴を1行]。「[特徴]」を武器に[元婚約者名]に取り入った女。
　[手口の本質を1行]。

◆ [新相手フルネーム]（[年齢]歳／[肩書]）
　[家柄や肩書を1行]。「[異名]」と呼ばれる[象徴]。
　[キャラの魅力・主人公との対比を1行]。

━━━━━━━━━━━━━━━━━━━
```
````

## 紹介文の書き方ルール

### 共通

- **1キャラ3行**（プロフィール行 ＋ 説明2行）
- 1行は約30〜45字、視覚的に揃える
- 結末・ざまぁの内容は絶対に書かない（フックだけ）
- 視聴者に「この人、どうなるの？」を持たせる

### キャラ別の役割

| 役割 | 書き方の方向性 |
|---|---|
| 主人公 | 凛々しさ・有能さ・誇りを提示。最後に「が、――」で含みを残す |
| 元婚約者 | 元の格の高さ／成り上がりの凄み を見せた上で、「歪み」「変質」を示唆 |
| ヒロイン悪役 | ビジュアルの可憐さ＋「あざとさ」の手口を客観的に描写。視聴者にイラッとさせる |
| 新相手 | 異質な強さ・主人公との対比・「この男いいかも」と思わせる魅力 |

### 順序（厳守）

**主人公 → 元婚約者 → ヒロイン悪役 → 新相手**

物語の登場順＝視聴者の感情の流れに沿う：
「主人公の凛々しさ → 元婚約者の歪み → 悪役のムカつき → 新相手への期待」

## 入力テンプレート

ユーザーに以下を確認：

```yaml
target_video: ""
# 例：「【ルクレツィア①】真実の愛と言い張る色ボケ騎士団長」
# シナリオファイルのパスでもOK

style: "short"
# short: 1キャラ3行（デフォルト）
# medium: 1キャラ4〜6行（特徴と関係性も含める）
# with_quote: 名言入り（特徴＋セリフ引用で世界観を演出）

scope: "main_only"
# main_only: 主要4人のみ（デフォルト）
# all_appear: シナリオに登場する全員
# include_family: 主要4人＋家族（背景人物も含める）
```

デフォルト（short × main_only）が一番YouTubeの概要欄向き。

## 保存先・ファイル名

- ファイル名：`【主人公名①】登場人物紹介.md`
  - 例：`概要欄/【ルクレツィア①】登場人物紹介.md`
  - 例：`概要欄/【エカテリーナ①】登場人物紹介.md`

---

## 良い例（参考）

`概要欄/【ルクレツィア①】登場人物紹介.md`、
`概要欄/【エカテリーナ①】登場人物紹介.md` を参照。

特に **オンブル・クロワゼット** は複数動画に共通登場するため、
紹介文は基本コピペで統一する（取り入る相手の名前だけ差し替え）：

```text
◆ オンブル・クロワゼット（19歳／没落男爵令嬢）
　黒髪ストレートに琥珀色の瞳。「か弱さ」を武器に[男の名前]に取り入った女。
　上目遣いと潤んだ瞳で、男の庇護欲だけを的確に撃ち抜く。
```
