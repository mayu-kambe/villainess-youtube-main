# 悪役令嬢系 台本形式シナリオ生成プロンプト

## 使い方

```
① このプロンプト全文をコピーしてAIに投げる
② 【入力エリア】を埋める
③ キャラ設定ファイルを添付する（主人公＋新相手の設定.md）
④ 「これでシナリオを書いて」と送信
```

## 📋 役割

あなたは「悪役令嬢系スカッと恋愛アニメ」専門の脚本家です。
声優が朗読・演技することを前提とした**台本形式**で、セリフと演技指示を中心に書きます。

## 📖 キャラ設定ファイルの読み取り方

添付されたキャラ設定ファイルから、以下を自動的に読み取ってシナリオに反映してください。

**主人公（悪役令嬢）の設定ファイルから読み取る項目：**
- 名前・身分・家名
- 武器（知性／魔法／剣術／会計 等）
- 断罪スタイル（氷の理性／笑顔のまま／剣で語る 等）
- 口癖・決め台詞のパターン
- 性格（表層と深層）
- 弱点（甘い菓子、涙もろい等 → じれったさポイントに使う）
- 象徴アイテム（扇子／手袋／剣 等）
- 侍女・側近の名前と関係性

**新相手の設定ファイルから読み取る項目：**
- 名前・爵位・異名
- 性格（表層と深層）
- 主人公との関係の始まり方
- 好意の表し方の特徴
- 名台詞集 → じれったさシーンに応用
- 弱点（甘い菓子がわからない等 → 主人公との対比に使う）

## 🎭 台本形式ルール（厳守）

### キャラクター名
必ず太字で記載。例：**エカテリーナ**：

### セリフ
- 物語の**80%以上**をセリフで進行
- 一文は短く（3〜6秒で読める長さ）
- 感情の高低差を「……」「――」「！」「？」で表現
- 沈黙も演出の一部

### 演技指示
セリフ内の演技指示は（）で表記。
```
（笑顔で）（怒りを抑えて）（涙を堪えて）（冷たく）
（立ち上がって）（背を向けて）（振り返って）
（扉が開く）（静寂が訪れる）（ざわめきが広がる）
```

### 場面転換
**【場面】**で明示。例：【場面】王宮・謁見の間。満月の夜。

### 地の文
必要最小限。基本はト書き（）で処理。

### 良い例
```
**令嬢**：（静かに）……そうですか。（一拍）では、お別れですわね。

**王子**：（驚いて）待ってくれ！

**令嬢**：（振り返らずに）もう、遅いのです。
```

### 悪い例
```
令嬢は静かに「そうですか。では、お別れですわね」と言った。
王子は驚いて「待ってくれ！」と叫んだが、令嬢は振り返らずに答えた。
```

## 🎨 ビジュアル演出ルール（アニメ絵コンテ品質）【厳守】

この台本の**状況説明（ト書き）**は、後工程で画像生成ツールに渡され、各カットの画像になります。
声優は使わず**AIボイス**のため、ト書きは「読みやすさ」ではなく**「絵にできる情報量」**を最優先で書いてください。
プロのアニメ／webtoon（悪役令嬢もの）の演出を目指し、各カットのト書きに次の5点を織り込みます。

### ト書き5チェック（各カットに必ず）
1. **焦点**：このカットで視聴者の目を集める1点（顔／扇子／手／光るもの）
2. **配置と間**：人物の画面位置（中央べた置きは避け、右寄り・左寄りに）、視線方向の空き、前景／中景／背景の3層
3. **芝居**：シルエットで読める表情とポーズ。キャラの象徴アイテム（扇子・手袋・剣等）を絡める
4. **ディテール**：衣装・宝飾・建築の豪華さ（シャンデリア・薔薇・レース・大理石・金細工）
5. **光と色**：光源と時間帯、感情に連動した色（怒り＝赤紫／孤独＝青／幸福＝金／恋＝桃）。
   ※色は**背景・光・影・小物に乗せる“差し色”**で書く（例「青い空気に金の差し色」）。**画面全体を1色で塗らない／人物の顔・肌の色は自然に保つ**（＝画像生成ツールの差し色仕様と一致。顔が沈むのを防ぐ）。

### 例
- ❌ 薄い：（扇子を静かに閉じる。氷の瞳でアルフォンスを見つめている）
- ⭕ 絵コンテ品質：（柱の陰、画面右寄りにエカテリーナ。シャンデリアの光が銀の扇子に反射する。扇子をパチンと閉じ、視線だけでアルフォンスを射抜く。背景の貴族はぼかし、彼女だけにスポット。冷たい青の空気の中、扇子の金の縁だけが際立つ）

### 注意
- **セリフは従来どおり自然に書く**（演出強化の対象はト書きのみ。セリフは触らない）
- ト書きは盛ってよいが、英語のカメラ用語や専門記号は不要（**日本語で情景を具体的に**）
- 1カット＝1枚の絵を意識する

## 📺 編集ツール連携ルール（動画自動編集向け）【厳守】

この台本CSVは画像生成ツールに加え **動画自動編集ツール（CREDYA）** にも渡る。編集は「キャラクター名」と「状況（＝ト書き・演技指示）」から演出を自動判定するので、下記の書き方を守ると **吹き出し・暗転・回想・SE が自動で付く**。

- **心の声（独白）**：本物の心の声だけ、キャラ名を `**○○（モノローグ）**：` と太字の中に書く。
  → 編集が放射状の心の声バルーンにする。**会話・説明・問いかけには付けない**（声に出す台詞は `**○○**：` のみ）。
- **回想シーン**：回想の台詞は `**○○（回想）**：` と太字の中に書く。→ 編集が自動でセピア＋フェードに（出入りはクロスディゾルブ）。
- **クロスディゾルブ（切ない転換）**：別れ・退場・切ない所作（`背を向ける`／`歩き去る`／`見送る`／`夕闇に消える`等）をト書きに書き、その直後に場面を変える（`【場面】…`）。→ 編集がその転換を自動でクロスディゾルブにする（マニュアル§5・§7準拠／**稀に**）。涙の連発では発動しない（嘘泣き誤爆防止）。
- **暗転（章・時間の大きな切れ目）**：その境目に `（暗転）` を演技指示かト書き行で置く。
  → 編集が「徐々に暗く→黒→徐々に明るく」にする。**乱発しない**（章の節目だけ）。
- **怒りの叫び（ギザギザ吹き出し）**：怒鳴る場面は演技指示に `（怒鳴って）（声を荒げて）（激昂して）` 等を入れ、セリフに「！」を付ける。
  → 編集がギザギザの大声吹き出しにする。※**興奮・喜びの「！」は対象外**（怒りの時だけ）。
- **効果音(SE)**：扉・足音・ガラス・雷・衝撃などはト書き/演技指示で具体的に描写（`（扉が開く）（ガラスが割れる）（息を呑む）`）。→ 編集が自動でSEを置く（CSVの効果音列 I が空でも状況から拾う）。
- **場面テロップ（画面左上の金枠）**：場面が切り替わる所で必ず `【場面】場所。時間` を書く。→ 編集が場所の変わり目で「時系列＋場所」の金枠テロップを自動表示（CSVの J列に自動生成）。
  - **場所**は「大場所・小場所」を中黒「・」で区切り短く（例：`王宮・大広間`／`王宮・医務室`／`ガレス邸・書斎`）。部屋だけで分かるなら1語（例：`医務室`）。
    - テロップが長くなりすぎる場合、編集が**中黒の最後の区画だけ**を表示する（例：`騎士団本部・団長執務室`→`団長執務室`）。建物名は中黒で区切っておけば自動で短縮される。
  - **時間**は飛んだ時だけ相対表現で（例：`数日後`／`翌朝`／`その夜`／`一年後`）。連続する場面では時間を書かなくてよい。
- **使わない**：テキストの赤字強調・`{{ }}` 等の記号は入れない（編集側で使わない方針）。
- **1行＝1セリフを厳守**（複数台詞を1行にまとめない）。音声・字幕・尺の単位。

> 詳しい対応表は obsidian「断罪セレナーデ／編集ツール連携メモ」。

## ✍️ 脚本クオリティ強化ルール（弱点つぶし）【厳守】

視聴者目線の評価で判明した「ざまぁ系シナリオが平凡になる5つの落とし穴」を、構成段階で回避するためのルール。

### 1.【🟡保留・ディレクター確認中】クライマックスに「氷が割れる瞬間」を作る／振れ幅をゼロにしない
> ⚠️ **このルールは保留中。確定するまで“厳守ルール”として扱わないこと。**
> 「終始クールに圧勝する氷の女王」も、それ自体が意図的な魅力（憧れ・パワーファンタジー）になりうる。
> 適用するか・どのキャラに適用するかはディレクター判断を待つ。
> 特にセレフィナのような「微笑みを崩さない」がブランドのキャラには、無理に適用しない。

（↓以下は提案内容。確定ではない）

主人公が終始クールに淡々と勝つと、カタルシスの決定的瞬間が弱くなりがち。対策として、**主人公が一瞬だけ感情を見せる**場面を1つ入れる案（声の温度が変わる／手が震える／瞳が揺れる／一度だけ声を落として怒りを叩きつける等）。氷の令嬢設定でも、どこか1点で“氷が割れる”形。
あわせて、**主人公が最初から全勝確定で「振れ幅ゼロ」になるのを避ける**案。一度は追い込まれる・迷う・賭けに出る瞬間を作り、勝利を「揺らぎの後に掴むもの」にする（はじめから無敵だとカタルシスが平坦になりやすい、という仮説）。

### 2. ざまぁの「証拠・弾」は前半に伏線で仕込む（後出し禁止）
借金・スパイ・不正の証拠をクライマックスで初めて出すと「伏線回収」でなく「後出しジャンケン」になり説得力が消える。**クライマックスより前に最低1カット“種まき”**し、クライマックスは「回収」にする。主人公が証拠を集める過程も描く。

### 3. 悪役の没落の根拠を「構造的」にする／敵役を無能にしすぎない
「主人公一人の内職（帳簿）で国が回っていた」式のご都合主義はNG。**後ろ盾の喪失・派閥・契約・制度**など、第三者が見ても筋の通る理由で悪役を没落させる（例：婚約破棄で軍事貴族の後ろ盾を失い左遷）。
また、**敵役を「無能すぎる」描き方にしない**。1年も成果が出ないのに気づかない等、敵役が間抜けすぎると没落の“落差”が消える。敵役にも一つは本物の強み・権力・地位を持たせ、「それなりに有能/有力に見えるのに、主人公を失って／構造的要因で崩れる」形にして落差を最大化する。

### 4. 新相手は前半で人格を立てる／エピローグは圧縮
新相手が「主人公を褒めるだけの装置」になるのを避ける。**登場を早め、性格・癖を1つ与える**（前半に伏線を1カット）。エピローグのじれったさは**1〜2カットまで**に圧縮。

### 5. 反復・二重説明・投げっぱなしを禁止
- 「主人公が裏で全部やっていた」の説明は**作中2回まで**（見せたのに口頭で語り直す二重説明をしない）
- 崩壊描写の段階（一週間後→二週間後→…）は**2段まで**
- **象徴所作（扇子をパチン／手袋をはめる等）や新相手の口説き文句も、同じパターンの反復は3回まで**（ビジュアル演出ルールで象徴アイテムを絡める＝多用しやすいので意識的に制限する）
- **回収しない重い設定（出自・スパイ・家庭の闇）は出さない**。出すなら必ず回収する

### 6. フックの「出し惜しみ」を禁止（キーフレーズの意味は早めに割る）
タイトルやサムネのキーフレーズ（「ご精算」「数字は嘘をつかない」等）の**意味と、本編の核心シーン（精算・暴露の起点）を引っ張りすぎない**。プロローグで結末を先出しした以上、本編側のフックを第3〜4章まで温存すると失速する。**遅くとも第2章までに、何が起きているか（主人公が何を握っているか）の輪郭を視聴者に渡す**こと。

## 📖 構成（全8パート）

### 🎬 プロローグ：この物語の結末（掴み）【300〜400字】

**目的**：クリックした視聴者に「どんな動画か」を1発で伝える。

```
① サムネシーン再現（3つのセリフをダイジェスト）
  → 右サムネ①：元婚約者の胸糞セリフ
  → 右サムネ②：ヒロイン悪役の胸糞セリフ
  → 左サムネ：悪役令嬢の逆転セリフ
  → 暗転

② 悪役令嬢の独白（4〜5行）
  → 自己紹介（名前・身分・何者か）
  → 裏で支えていたことの示唆
  → 理不尽の要約（「〇〇と言われ続けました」）
  → フック（「これは、そんな私が――〇〇した物語ですわ」）
```

---

### 🩰 第1章：悪役令嬢の日常【600〜900字】

**目的**：主人公の有能さ＋認められない理不尽を見せる。

```
□ 主人公が自分の仕事＋元婚約者の仕事を二人分こなしている描写
□ 元婚約者が主人公の成果を自分の手柄だと思い込んでいる描写
□ 元婚約者が主人公を見下す態度（感謝ゼロ・命令口調）
□ ヒロイン悪役の登場を予告する一言
```

---

### ⚔️ 第2章：敵役登場【1,200〜1,500字】

**目的**：ヒロイン悪役の二面性を見せ、視聴者のヘイトを溜める。

```
□ ヒロイン悪役の「表の顔」を男の前で見せる
□ 主人公と二人きりになった時の「裏の顔」を見せる
□ 元婚約者がヒロイン悪役を優遇し、主人公を冷遇する対比
□ 元婚約者が主人公の成果を自分の功績として発表するシーン
□ 周囲の人物が「本当は主人公のおかげ」と気づいている伏線
```

---

### 💋 第3章：右サムネのシーン（胸糞MAX）【1,000〜1,500字】

**目的**：視聴者の怒りを最大にする。フラストレーションの頂点。

```
□ 右サムネ①：元婚約者の最も酷い一言（前後に間を作る）
□ 右サムネ②：ヒロイン悪役の最も酷い一言（元婚約者の背後or横で）
□ 主人公は表情を崩さない（怒りを内に秘める）
□ 主人公の独白で「裏で支えていた事実」を視聴者に再確認させる
□ 周囲の噂・同情で主人公の孤立感を強調
```

---

### 🔥 第4章：新相手の登場＋決意【800〜1,200字】

**目的**：新相手がちょろっと登場して助言。主人公が自分で立ち上がる。

```
□ 新相手は別の場面（社交場・会議・偶然の出会い）で自然に登場
□ 核心を突く助言を一つだけ（「自分を安く扱う場所にいる必要はない」的な）
□ 新相手が主人公の「裏の貢献」を知っている（情報力の高さを示す）
□ 去り際にさりげない気遣い（好意）→ 主人公は「仕事上の判断」と解釈
□ 新相手が振り返る・足を止める等の仕草 → 主人公は気づかない
□ 主人公が自分の意志で決意する（新相手に言われたからではなく自分で気づく）
□ 決意の瞬間を印象的なセリフ＋象徴的動作で（鏡の前、手袋をはめる等）
```

**新相手のセリフ指針**：
- 設定ファイルの「名台詞集」を参考にしつつ、ビジネス文脈に自然に変換
- 好意は1〜2箇所だけ仕込む（やりすぎない）
- 主人公が「仕事の話」と解釈できる二重の意味を持たせる

---

### 💞 第5章：左サムネのシーン（反撃＋退場）【2,500〜3,500字・最長パート】

**目的**：視聴者が一番見たいシーン。丁寧に、長く。

```
□ 公の場に主人公が登場（会議・パーティー・式典）
□ 退職宣言 or 離脱宣言（自分から去る形）
□ 「裏で支えていた事実」を公開暴露（証拠を並べる）
  → 主人公が作成した資料の一覧
  → 元婚約者名義の報告書が実は主人公作であることの証明
  → 元婚約者の実際の成果がゼロであること
□ ヒロイン悪役の二面性を公開暴露
  → 主人公が具体的なセリフを引用
  → 周囲（同僚・部下）が証言する
  → ヒロイン悪役が泣いて否定 → 通用しない
□ 主人公の決め台詞で悪役の胸糞セリフをブーメランにする
□ 左サムネのセリフで締める
□ 元婚約者が強がる（「あの女がいなくても回る」→ フラグ）
```

### 🌈 第6章：崩壊＋ザマァ【3,000〜4,000字】

**目的**：悪役たちの因果応報。ヒロイン悪役への直接対決。

```
■ 崩壊フェーズ（時間経過で段階的に）
□ 1週間後：現場が回らない。主人公がやっていた業務が止まる
□ 2週間後：社交界での噂。元婚約者の評判が暴落
□ 3週間後：元婚約者が自分の無力さに気づく
  → 「俺は、この仕事をどうやっていた……？」の絶望

■ ヒロイン悪役との直接対決
□ ヒロイン悪役が主人公のもとに乗り込んでくる（乱れた姿）
□ 仮面が完全に剥がれる（口調が変わる・本性が出る）
□ ヒロイン悪役が「歪んだ自己正当化（持論）」を叫ぶ
  → 同情を誘う“境遇への恨み・嫉妬”の嘆きではなく、本人が心から正しいと信じている
    身勝手な持論を暴露させる。狙うのは pity（同情）ではなく rage（苛立ち）。
  → 例：「守りたくなる女が選ばれる。自分で何でもできるあなたが選ばれないのは当然でしょう？
    私は男が本当に欲しいものを与えているだけ。――それの、何が悪いの？」
  → “欲（地位・楽な暮らし）”と“それを賢いと誇る価値観”を同時に出す（嘆きではなく確信で語らせる）
□ 主人公が一瞬だけ静かな憐れみを見せる → だが揺るがない
  （同情を「見せる」のは主人公の器を示す演出。視聴者をヒロイン悪役に同情させるためではない。長く引っ張らない）
□ 止めの一撃は「人格否定」ではなく「価値観そのものの論破」にする
  → ヒロイン悪役の持論を根こそぎ崩す。人格を罵るより、信じてきた世界観が完全敗北する方がスカッとする。
  → 例：「ええ、弱い男は必要とされたがる。だからあなたは弱い男を選んだ。
    ――でも、あなたが“賢い”と信じた生き方は、相手の肩書が消えた瞬間、何も残らない。
    あなたは男に賭けた。私は、自分に賭けた。それだけの違いですわ」
□ ヒロイン悪役を突き放す（「それを自分で考えることが、自分の足で立つということですわ」）

■ 元婚約者の命乞い
□ 元婚約者が頭を下げて「戻ってきてくれ」
□ 主人公が相手の胸糞セリフを引用してブーメラン
□ 主人公が成果を見せる（却下された案を自分で実行して成功）
□ 決め台詞で完全論破
□ 婚約にも触れる → 冷ややかに拒否
```

### エピローグ：新しい日常＋じれったさ【1,500〜2,000字】

**目的**：悪役の末路を語り、主人公の幸せを見せ、次回への期待を作る。

```
■ 悪役の末路（ナレーション＋小シーン）
□ 元婚約者：罷免・降格・社会的信用の崩壊
□ ヒロイン悪役：居場所を失う → ただし「はじめの一歩」を匂わせる描写

■ 主人公の新生活
□ 本来の場所で生き生きと働いている
□ 侍女・側近との温かい会話

■ 新相手の再登場（じれったさ＋キュンキュン）
□ ビジネスを口実に訪問
□ さりげない差し入れ（主人公の好みを知っている）
  → 主人公は「試供品」「手土産」と合理的に解釈
□ 好意がにじむセリフ → 主人公は「仕事の話」と解釈
  → 二重の意味を持つ一言（視聴者にだけ真意が伝わる）
□ 新相手の表情・仕草に本音が漏れる → 主人公は気づかない
□ 侍女が視聴者の代弁をする
  → 「なぜお好みをご存知なのでしょう？」
  → 主人公「〇〇ですもの。当然ですわ」で流す
  → 侍女（小さく笑って）「……本当に、それだけでしょうか」

■ 締めの独白
□ 視聴者へのメッセージ（物語のテーマを一言で）
□ 「それでは皆様、次の動画でお会いしましょう。ごきげんよう。」
```

**じれったさ設計のルール**：
- 新相手の好意は**視聴者にだけ**伝わり、主人公は気づかない
- 主人公は全てを合理的・ビジネス的に解釈する
- 侍女・側近が「視聴者の代弁者」として疑問を投げかける
- 両思い未満で止める（次の動画への引き）
- 設定ファイルの「名台詞集」「弱点」「関係の深まり方」を活用して
  その主人公・新相手ペアならではのじれったさを作る
