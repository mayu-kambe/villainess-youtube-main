# シナリオ評価：視聴者目線フィードバック（2026-06-07）

悪役令嬢系YouTube（AIボイス＋静止画スライドショー、ざまぁ／スカッと系）として、既存シナリオを**辛口・視聴者目線**で評価したもの。
今後の生成プロンプト改善の材料にする。

## 評価軸（7つ）
1. フックの強さ（冒頭で掴めるか）
2. 中盤の冗長さ（同じ情報・展開の繰り返し）
3. 独白過多（内面モノローグでテンポが死んでいないか）
4. ざまぁの説得力（悪役の没落の根拠が薄い／ご都合主義でないか）
5. 投げっぱなしの重い設定（回収されない伏線・唐突な重設定）
6. 新相手が薄い／エピローグが長すぎないか
7. 感情の爆発（カタルシスの決定的瞬間があるか）

## 🔴 横断チェック：全シナリオ共通の弱点パターン

個別ではなく**生成テンプレ由来の構造的クセ**。同じ弱点が繰り返し出ている。

| 共通の弱点 | 該当 | 深刻度 |
|---|---|---|
| **主人公の「感情の爆発」がない**（終始クールに淡々と勝つ＝カタルシスの決定的瞬間がない） | エカテリーナ①／愛の代償／氷の令嬢／セレフィナ | 最重要 |
| **ざまぁの「弾」が後出し・ご都合主義**（証拠・借金・スパイを全部クライマックスで初出し） | エカテリーナ系3本で顕著 | 最重要 |
| 新相手が薄い＋エピローグが長い（新相手が“主人公肯定装置”） | 愛の代償／氷の令嬢／セレフィナ | 中 |
| 中盤の反復・二重説明（「裏でやってた」を3〜5回／見せたのに語り直す） | ほぼ全部 | 中 |
| 重い設定の投げっぱなし（母を殺した／スパイ網／階級格差を出して放置） | エカテリーナ①／愛の代償／氷の令嬢／セレフィナ | 中 |

### ✅ ルクレツィア①が克服している点（＝目指す形）

- 主人公に「氷が割れる瞬間」を作る（壁殴る／赤面／瞳が揺れる）

## 🛠 生成プロンプトへの反映提案（やれば全シナリオが底上げ）

「🎬 悪役令嬢系台本形式シナリオ生成プロンプト.md」に以下のルールを足す：

1. **クライマックスに「氷が割れる瞬間」を1つ必須化** — 主人公が一瞬だけ感情を見せる（声の温度が変わる／手が震える／瞳が揺れる）。終始クール禁止。
2. **ざまぁの証拠は前半に伏線として仕込む（後出し禁止）** — 借金・スパイ・証拠は、クライマックスより前に最低1カット“種まき”する。クライマックスは「回収」にする。
3. **没落の根拠を構造的にする** — 「主人公一人の内職で国が回っていた」式のご都合主義を避け、後ろ盾喪失・派閥・契約など筋の通る理由にする。
4. **新相手は前半で人格を立てる／エピローグは圧縮** — 新相手の登場を早め、性格を1つ与える。エピローグのじれったさは1〜2カットまで。
5. **「裏でやっていた」の説明は2回まで／重い設定は回収 or 削除** — 反復・二重説明を禁止。回収しない重い設定（出自・スパイ・家庭の闇）は出さない。
