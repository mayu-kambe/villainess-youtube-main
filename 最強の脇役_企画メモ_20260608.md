# 🎭 「最強の脇役」企画メモ（2026/06/08）

**これは何？** → メインキャラとは別に、**チャンネルの“顔”になる人気脇役**を1人作りたい、という企画。
人気ドラマ・映画・悪役令嬢webtoonを調査し、その結果をもとにキャラ案を提案します。

**狙い** → 悪役オンブルが「共通の悪役マスコット」なら、その対になる「**共通の善玉マスコット**」を作る。切り抜き映え・視聴者の代弁・ファンの愛着で、**チャンネルの集客＆ブランドの軸**にする。

## 📚 調査でわかったこと

### A. 実在ドラマ・映画の「人気脇役」7法則

| 法則 | 中身 | 代表例 |
|---|---|---|
| ① セリフが主役より強い | 説明台詞を背負わない分、一番キレた名言をもらえる | エドナ・モード（インクレディブル） |
| ② 尺より「存在感」 | 短い出番でも刺さる（ジョーカー33分／エドナ2シーン） | 「presence matters more than arc」 |
| ③ デッドパン毒舌 | 真顔で容赦なくズバッと言う | 毒舌同僚キャラ全般 |
| ④ 主役の対比役 | 主役が我慢する所を代わりに言う＝主役が引き立つ | — |
| ⑤ 一発でわかる記号 | 見た目・所作・決め技 | エドナの丸眼鏡 等 |
| ⑥ 決め台詞 | 毎回言う一言＝切り抜きで回る | ヨナの「オッパ！」 |
| ⑦ コメディの下に深さ | 笑える表面の下に情の層 | シュレックのドンキー |

**特に女性向けで鉄板の2型：**
- **辛口エレガントな“毒舌真実告げ役”**（ダウントン・アビーの伯爵未亡人／GoTオレナ／ブリジャートンのレディ・ダンベリー）
- **主人公を無条件で推す“ハイプ忠臣”**（ウ・ヨンウの親友）

### B. 悪役令嬢ジャンル（webtoon/漫画/アニメ）の実例 ←ここが重要

**「毒舌・忠実なメイド／側仕え」は、このジャンルでも“鉄板の推され役”でした。**

| 作品 | 人気脇役 | 刺さっている点 |
|---|---|---|
| 異世界の主人公になりました | **ペネロペ（侍女）** | 守護者＆相談役。**鋭い舌・物怖じしない態度** |
| 同上 | フェリックス（騎士） | 普段いじられ役だが**実は国最強級**。ギャップ |
| Beware of the Villainess! | **ヨナ（メイド）** | **物欲まみれで無遠慮**。「給料が高いから忠実」の正直さ |
| 私の推しは悪役令嬢 | クレアのメイド | 令嬢が一番信頼し、姉のように慕う |

**調査からの3つの気づき：**
1. **ジャンルの感情の核**＝「世間が“悪役”と誤解する令嬢を、**唯一“本当の価値”で見ている脇役**」。ウチの主人公（悪役と誤解された被害者）と完全に噛み合う。
2. **“正直な金銭動機メイド”（ヨナ型）**＝媚びない毒の笑い。
3. **“いじられ役だが実は最強の護衛”（フェリックス型）**＝メイドと組ませると凸凹コンビの化学反応が出る。

## 🎯 提案：最強の脇役「渡りの毒舌侍女」

**役柄＝侍女（専属メイド）。** 低い身分なのに一番“人を見る目”がある＝「最強のモブ」。
設定上「どんな令嬢でも立て直す伝説の侍女」として各家に派遣される形にし、**全シナリオに登場できる看板キャラ**にする。

### キャラ設計（調査の良いとこ取り）

| 要素 | 内容 |
|---|---|
| **核** | 鋭い舌・物怖じしない・絶対忠誠の侍女（ペネロペで実証済み） |
| **感情の核** | 世間が令嬢を“悪役”と誤解しても、**彼女だけは真価を見抜いて隣にいる**（「お嬢様を悪く言う口は、私が縫い留めますわ」的な絶対擁護） |
| **毒舌技** | 悪役オンブルのあざとさを**真顔で即・解説して破壊**（「ただ今の落涙、右目が0.5秒遅れました」） |
| **笑い** | 動機を**正直に毒で言う**（「忠誠ではございません。お嬢様の紅茶が世界一なだけですわ」） |
| **推し忠臣** | 主人公×新相手を誰より応援し、じれったさを実況（「で、いつ気づかれます？」） |
| **ギャップ** | 普段毒舌→令嬢の幸せの話で急にデレて崩れる |

### このキャラが「悪役オンブル」とどう噛み合うか
- オンブルの弱点は「**女には一発でバレるが、男は絶対に見抜けない**」。
- 男（元婚約者）は全員騙される。そこに**オンブルを即見抜いて斬る侍女**を置く。
- ＝視聴者の「**男ーー目を覚ませ！**」を、その場で代わりに晴らす存在になる。

### なぜバズると見込んでいるか（根拠）
**1キャラで3つの“バズ装置”を同時に持てるから：**
1. **デッドパン毒舌の一刀両断**＝切り抜き・Shortsで一番回る単位（主エンジン）
2. **視聴者の代弁**＝「私が言いたかったこと言ってくれた」で共有・コメントが伸びる
3. **ラブの実況ツッコミ**＝じれったさ回でコメ欄が盛り上がる

## 🤔 決めてほしいこと（2つ）

### 【脇役・決定1】キャラのトーン
- **X：若い・コミカル寄りの毒舌**（テンポ重視・Shorts向き・軽快）
- **Y：年配・辛口ベテランの“執事級”**（上品な毒・貫禄・重厚）
- **おすすめ：Xベース＋Yの貫禄を少し混ぜる**（若くてキレるが忠誠は老侍女級）

### 【脇役・決定2】相棒をつけるか
- **単体：** まず侍女1人で看板キャラを立てる
- **凸凹コンビ：** 最初から「侍女＋いじられ役だが実は最強の護衛騎士」で設計（対比の化学反応が出る）
- **おすすめ：** まず侍女単体で立て、護衛は後から追加でもOK
